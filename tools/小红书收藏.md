一句话，让AI帮你整理落灰的收藏夹
我不知道你小红书收藏夹里的笔记是不是收藏即积灰？反正我大多数笔记是
我自己平时探店也有个专门收藏夹有些地方已经去过有些已经过时
那么上百条笔记每个都要手动打开复制链接太麻烦了。想着能不能一键搞定所有链接的提取？
【工具选择:Claude Code】
可能你也听说过各种 Al 编程助手，但我选 Claude Code 的原因:
 使用Chromedev Tool Skills 可以直接控制浏览器：像真人一样点击、浏览网页
自动学习改进：反正我用Cursor它一直跟我反复确认 approve，但Claude自主性感觉更强，它会记住我的偏好，越用越聪明
【预先准备工作】
+ Chromedev Tool Skills+飞书文档MCP
【过程拆解】
我：帮我从小红书收藏夹提取20条笔记链接Claude:好的，开始提取...
Claude自己识别使用Chromedev Tool Skill 自行打开浏览器、导航到我的收藏夹页面。
我只需要帮它登录一下，它会自己"思考"并提取数据当然过程中它提取的链接不对，我开始调教它告诉它什么是正确的小红书笔记的链接
最终它成功爬取了帖子并直接以飞书文档发给我
【复盘和学习】Not Me! !
看看我们家Claude孩子在过程中的自己整理的复盘以及关键学习点
不要相信网页上的链接格式 
- 小红书有 /board/ (收藏夹内部)
- 有 /explore/ (页面导航)
- 只有 /discovery/item/ 才是可分享的 必须验证链接
- 提取后一定要点开看看
- 不要假设格式正确
当然我也不会浪费这一次宝贵的学习机会，通过learning skills它会记住它在过程中犯的错误并重新生成一个新的skills 记录到它的skills库里
而下一次如果我需要再次让它去我收藏夹里拿帖子它就会自动调用这个新技能
