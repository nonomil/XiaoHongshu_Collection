# 多平台文章归档 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在保留小红书深度能力的前提下，为微信公众号文章、知乎文章、CSDN 博客新增一个轻量的“文章型平台归档模式”，支持 URL 输入、正文提取、图片保留和 Markdown 导出。

**Architecture:** 复用当前项目已经存在的 Chrome/CDP 打开页面能力和 Markdown 落地思路，但不把新平台直接塞进小红书专用逻辑里。新增一套“文章源适配器”层，先统一 URL 分类、页面抽取和标准文章对象，再由独立的 `article_export` 模块输出 Markdown，这样一期只做正文归档，不承诺评论和平台深爬。

**Tech Stack:** Node.js、现有 CDP 连接逻辑、现有 `node --test` 测试体系、静态 HTML fixture、最少量共享导出工具。
