# 单笔记评论区采集 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为单笔记导出增加“全部评论采集 + 原始评论归档 + 有用评论总结与全文输出”能力，同时不阻塞正文、图片、OCR 主流程。

**Architecture:** 在 `cdp_note.js` 中补充评论提取函数，在 `note_export.js` 中增加评论归档、评论 Markdown 渲染和 AI 处理占位。主文件继续输出正文、OCR、图片，评论原文单独保存到 `_comments/<noteId>.json`。

**Tech Stack:** Node.js、Chrome CDP、原生 `node:test`、现有 OpenRouter/vision 配置链路。
