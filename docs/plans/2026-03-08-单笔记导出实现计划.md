# 单笔记导出 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 将单笔记导出正式收敛为独立入口，支持 URL、分享文本、短链和 `--current`，并在尽量复用现有链路的前提下稳定输出到 `单挑保存` 目录。

**Architecture:** 保留 `scripts/save_note.js` 作为独立入口，继续复用 `note_input.js`、`cdp_note.js`、`note_export.js` 三层能力。当前版本只补输入兼容、详情抓取稳定性、评论链路和错误提示，不引入新的浏览器运行时或 Cookie 持久化方案。

**Tech Stack:** Node.js、Chrome CDP（`ws`）、Tesseract.js、Node 原生测试运行器、现有 OpenRouter 配置链路
